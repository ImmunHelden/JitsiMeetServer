# Jitsi services deployment

```bash
  Usage examples:

  jitsi_ansible_wrapper --install
  jitsi_ansible_wrapper --start

  other options:
    -i, --install                  # installs all jitsi related services and docker containers
    -u, --start                    # starts all jitsi related services and docker containers  #todo
    -s, --stop                     # stops all jitsi related services and docker containers  #todo
   -rs, --restart                  # starts all jitsi related services and docker containers  #todo
    -d, --down                     # down all jitsi related services and docker containers  #todo
    -r, --remove                   # resets the host and removes everything jitsi related
        --dry-run                  # Do not make any changes, just print commands  # todo
    -v, --version                  # Show program version
        --debug                    # Enable debug output
```
